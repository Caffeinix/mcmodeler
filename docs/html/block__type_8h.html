Typedef Documentation

typedef int blocktype_t

Represents a block type, aka a block ID.

These do not correspond to the Minecraft block IDs, but are instead determined by the order in which blocks are read from the JSON.

Todo:
This may soon change to make save files more compatible. Minecraft's block IDs serve as a good reference that will not change.

Variable Documentation

const blocktype_t kBlockTypeAir = -1

Special block type ID for "air", or empty space.

This is used whenever it is necessary to generate a BlockPrototype for a block that does not exist.

const blocktype_t kBlockTypeUnknown = -2

Special block type ID for the "unknown block".

This is used to indicate that no block type is currently selected in the BlockPicker.
