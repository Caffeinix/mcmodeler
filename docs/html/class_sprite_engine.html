Detailed Description

Constructs and returns sprite images for blocks to represent the block in a 2D context.

The sprite images returned by the engine can take into account the block's geometry, orientation, and whether it is being drawn in the BlockPicker or in the LevelWidget.

QPixmap SpriteEngine::createSprite ( const Texture &  texture,
const BlockProperties &  properties,
const BlockOrientation *  orientation = BlockOrientation::noOrientation() 
)

Creates and returns a sprite pixmap that can be drawn to represent a block in 2D.

texture will be used as the basis for the sprite, with modifications made based on properties and orientation, if given. To obtain a sprite suitable for the BlockPicker widget, pass BlockOrientation::paletteOrientation() as orientation.

Note:
SpriteEngine caches pixmaps when possible, so this method is fairly inexpensive after the first time a particular block is seen in a particular orientation.
The caching assumes that a particular properties parameter implies the same texture each time.
