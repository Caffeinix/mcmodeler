Detailed Description

An abstract class representing a particular method of drawing blocks given one or more positions.

To use a Tool, repeatedly call setPositionAtIndex() until the tool has enough points to work with (how many points this is depends on the tool: for a line tool, the required number is two, whereas a flood fill tool would only need one). Then, call draw() and pass the block information describing what you want to draw, and a BlockTransaction you want the tool to populate.
