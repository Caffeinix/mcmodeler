Detailed Description

Represents a 2D texture that can be drawn onto 3D geometry.

There are two kinds of textures: those that are loaded directly from a single dedicated image file, and those that are loaded from a particular tile of a sprite sheet image.

Constructs a texture that draws a tile from a sprite sheet image at path, tinting it with color using a particular composition mode.

Parameters:
widgetThe OpenGL widget into which this texture can draw.
pathA filesystem or QResource path pointing to an image in a supported format.
x_indexA zero-based index indicating which column the tile we should draw is in.
y_indexA zero-based index indicating which row the tile we should draw is in.
x_sizeThe width of a single tile. This must be uniform, and there is no support for margins.
y_sizeThe height of a single tile. This must be uniform, and there is no support for margins.
colorThe color we should tint the texture before we render it.
modeThe composition mode we should use to fill the texture. QPainter::CompositionMode_Multiply is often a good choice here.
Note:
Special care is taken to maintain the alpha channel of the original pixmap when tinting, so you can safely use composition modes that would normally nuke the alpha channel (such as multiply).
